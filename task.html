Коротко про Python

Таким чином, python підходить для вирішення левової частки повсякденних завдань, будь то резервне копіювання, читання електронної пошти, або ж якась іграшка. Мова програмування Python практично нічим не обмежений, тому також може використовуватися у великих проектах. Наприклад, python інтенсивно застосовується IT-гігантами, такими як, наприклад, Google і Yandex. До того ж простота і універсальність python роблять його одним з кращих мов програмування.
